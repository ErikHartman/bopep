## Multi-objective vs single-objective

Single objective

```py
obj = {peptide: val, peptide: val} etc.
```

Multi objective

```py
obj = {peptide: {name:val, name:val}, peptide: {name:val, name:val}} etc.
```

Outputs replace the vals with tuples of mean,var
